# WELCOME TO HIGH SCHOOL!!

Six years ago now, I went into ninth grade. I'd went to a vocational school in place of my electives in high school. This course was Computer Information Science, which I stayed in for four years. I kind of regret taking the course; it did not get me anything industry-certified. Instead, it only gave me what would count as 4 years of "experience". However, I now have a pretty long list of programs that show my progression over the years. The games I made like YUP and Krueger Clicker near the tail end of these, which show that I got better with larger-scale projects.

## WHAT IS HERE?

Folders divide the projects by coding language, with subfolders diving by year.

## EXTRA NOTES

I was a teenager when I made these; a young teenager. Because of that, I was crude in humor and integrated my humor with my projects. There will be various random jokes and nonsensical bits thrown in, but I chose to leave them here because I feel as if they reflect my growth and improvement. 

In one project in particular, there are various pieces of AI-generated imagery. I don't really like AI all that much, which is why it baffles me that it's there in the first place, but I chose to leave them out. Unless it was *specifically* for an assignment, which I specify in the assignment, the code is AI-free. 

There are a lot of bad programming practices in here. Imports in functions, functions that should be integrated into classes, sometimes me even being flat-out wrong with concepts like encryption. However, this is not a bad thing; these are old pieces of code that show my earlier years of programming and how I improved at it.

Soon, I hope to get into full-stack web development. I want to code bigger and better things, and these are the earliest stepping stones.
